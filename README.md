## Explica las 3 formas de usar CSS en HTML
Nosotros utilizamos estas tres formas:
CSS interno-
Dentro del <head></head> utilizamos type="text/css"


CSS externo-
<html>
<head>
    <title>Título de la página</title>
    <style type="text/css">
        div {
            background:#FFFFFF;
        }
    </style>
</head>


CSS Embebido-
Con la etiqueta <style></style>
